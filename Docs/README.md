# 🎬 Cinespoilers API

API REST desarrollada con **Django Rest Framework** para gestionar películas mediante operaciones CRUD.

## 🚀 Tecnologías usadas

- Python
- Django
- Django REST Framework
- SQLite
- Thunder Client

## 🔗 Endpoint principal

`GET | POST | PUT | DELETE`

`http://127.0.0.1:8000/api/peliculas/`
